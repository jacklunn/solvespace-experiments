# SolveSpace Experiments

Parametric 3D CAD models built with [SolveSpace](https://solvespace.com), an open-source constraint-based modeler.

## Models

| File | Description |
|------|-------------|
| `threaded-container.slvs` | Parametric threaded container body |
| `threaded-container-top.slvs` | Matching threaded lid/top for the container |

## Requirements

- [SolveSpace](https://solvespace.com/download.pl) 3.x

## Usage

Open any `.slvs` file directly in SolveSpace. Models are fully parametric — edit the sketch constraints to resize.

## Notes

These models are experiments in designing printable threaded assemblies. The thread profile is designed to be 3D-print-friendly with generous tolerances.
